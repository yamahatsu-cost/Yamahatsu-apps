# 楽天お得比較ベース

楽天市場の商品を最大5件までまとめて検索し、価格・ポイント倍率・送料補正・クーポン補正を使って比較するフロントエンドベースです。

## この版でできること

- 商品名を最大5件まで入力して検索
- 楽天APIから候補商品を取得
- 商品画像、価格、ポイント倍率、ショップ名、リンクを表示
- 実質価格 = 商品価格 - ポイント評価 - クーポン補正 + 送料補正
- 商品ごとに候補を選択
- 上位候補から「同一ショップは送料1回」前提で最安組み合わせを計算
- 検索条件とAPIキーを localStorage に保存

## この版でまだ手入力が必要な部分

- クーポン金額
- 送料の正確な金額

楽天の商品検索APIでは、価格・ポイント倍率・ショップURL・画像・送料無料フラグは取得できますが、クーポンの自動取得や送料金額の完全取得は標準検索だけでは難しいため、この版では補正入力にしています。

## 使い方

1. `index.html` をブラウザで開く
2. 右上の「API設定」で Rakuten Application ID と Access Key を保存する
3. 商品名を入力して「まとめて検索」
4. 候補の送料補正・クーポン補正を必要に応じて修正する
5. 「最安組み合わせを計算」でまとめ買い候補を確認する

## 楽天APIの準備

Rakuten Web Service でアプリを作成し、Application ID と Access Key を取得してください。

## 今後の拡張候補

- 商品ページ解析による送料金額の自動補正
- クーポンの自動読取
- 同一ショップの送料無料条件対応
- 検索候補のキャッシュ
- 除外ブランド・除外ショップの高度設定
- バックエンド化してキーをブラウザに置かない構成

## 注意

- Application ID と Access Key はこの版ではブラウザの localStorage に保存されます。
- 本格運用する場合は Node.js や Cloud Functions などで API プロキシを作る構成を推奨します。
